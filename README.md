# Utils
[![npm version](https://badge.fury.io/js/utils.svg)](https://badge.fury.io/js/utils)

A collection of pure Typescript utility functions.

## Installation
You can install the package via npm:

```bash
npm i utils
```

## Usage

```javascript
import * from 'utils';
```
